Conexion Suprema

Una colección de aplicaciones móviles para profesor y alumno, que fungen ambas como coomplemento una de la otra. Las aplicaciones móviles y web se abastecen de la base de datos para poder llevar a cabo sus funciones de manera correcta y dinámica.

Nuestros sensores inteligentes se conectarán directamente a la base de datos, abastenciéndo cada una de sus tablas relacionadas e interconectadas con las aplicaciones.
